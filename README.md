# Project Installation 📝

### Start with cloning the repository and navigate into it

```bash
git clone https://github.com/Bakhaw/test-prisma.git
cd test-prisma
```

### Using npm, install the module dependencies

```bash
npm install
```

<br />

# Project Usage 🔮

### Start the project and you now have access to it in your favorite browser 😁

```bash
// Using http-server

npm start
👉 visit http://localhost:8080

----------------------------------------

// Using webpack

npm run build
👉 open index.html inside your browser
```
